HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user.
  • Used to create the structure of a webpage.
  • It is made up of elements which work together to provide content and the basic framework for all pages on the web.
  • An element is a type of content on the webpage.

    • Commonly used elements are

  • Text elements
  • Image elements
  • Header elements
  • List elements

    • Common element types

  • Elements are also used to divide the content on the page to describe the content to browsers and other developers
  • Some common element types used to divide and describe a page
  • Meta-related elements
  • Content division elements
  • Semantic elements

    • Head and Body elements

  • Two elements are important to understanding how the structure works. The head and body element
  • The Head element contains information (metadata) that helps browser to render the page correctly
  • The Body element contains the content that is shown to the user

    • Metadata tags

  • title element defines the webpage. This element is extremely important for SEO
  • style element defines the CSS styles
  • base element defines the the base URL for webpage
  • link element connects an external resource to the HTML document
  • meta element defines metadata such as character set, description, keywords, author, and viewport
  • script element embeds client-side scripts in the HTML document

    • Nerd Notes

  • Elements nested inside other elements are called child elements. The element that holds all the children is the parent element
  • element and tag are used interchangeably in the web development world, but there is a difference. An element refers to the entire element from opening to colsing tag, whereas a tag refers to what appears inside the angle brackets.
  • Headers: (h1) to (h6) represent level of heading.
  • The (ul), (ol), and (li) elements represent unordered list, ordered lists, and list items. Basically represented as bulleted or numbered lists.
  • footer element appears at the bottom of the page and usually contains the author, copyright, contact, sitemap, and navigation.
